# ShopSphere

A free-tier friendly e-commerce project built with ASP.NET Core Web API, Vue 3, Tailwind CSS, and Supabase PostgreSQL.

## Tech Stack

- ASP.NET Core Web API
- Vue 3
- Tailwind CSS
- Supabase PostgreSQL
- Entity Framework Core

## Current Features

- Product listing API
- Product details API
- Supabase database connection
- Vue product listing page
- Tailwind UI setup
